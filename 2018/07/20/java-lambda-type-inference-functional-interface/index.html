Java Lambda (2) нғҖмһ… м¶”лЎ кіј н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨ

нғҖмһ… м¶”лЎ  Type Inference

нғҖмһ… м¶”лЎ мқҙлһҖ нғҖмһ…мқҙ м •н•ҙм§Җм§Җ м•ҠмқҖ ліҖмҲҳмқҳ нғҖмһ…мқ„ м»ҙнҢҢмқјлҹ¬к°Җ мң м¶”н•ҳлҠ” кё°лҠҘмһ…лӢҲлӢӨ. мһҗл°”м—җм„ңлҠ” мқјл°ҳ ліҖмҲҳм—җ лҢҖн•ҙ м°Ёмһ… м¶”лЎ мқ„ м§Җмӣҗн•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ мһҗл°”м—җм„ңмқҳ нғҖмһ… м¶”лЎ мқҖ м ңл„ӨлҰӯкіј лһҢлӢӨм—җ лҢҖн•ң нғҖмһ… м¶”лЎ мқ„ л§җн•©лӢҲлӢӨ.

м ңл„ӨлҰӯм—җ лҢҖн•ҙм„ңлҠ” мһҗл°” 7м—җм„ң лӢӨмқҙм•„лӘ¬л“ң м—°мӮ°мһҗ(<>)лҘј мқҙмҡ©н•ҙм„ң нғҖмһ…мқ„ л„ҳкІЁмЈјм§Җл§Ң мһҗл°”к°Җ м¶”мёЎн•ҳкё°м—” н•ңкі„к°Җ мһҲмҠөлӢҲлӢӨ. мһҗл°”мқҳ м»ҙнҢҢмқјлҹ¬лҠ” Type Erasure лҘј мӮ¬мҡ©н•ҳлҠ”лҚ°, мқҙлҠ” м»ҙнҢҢмқјн•  л•Ң нғҖмһ… м •ліҙлҘј м ңкұ°н•©лӢҲлӢӨ.

мӢӨн–ү мӢңк°„м—җлҠ” лӘЁл“  кІғл“Өмқҙ Object мқҳ мқёмҠӨн„ҙмҠӨлЎң л„ҳм–ҙк°Җкі  мқҙл©ҙм—җм„ң м Ғм Ҳн•ң нғҖмһ…мңјлЎң мәҗмҠӨнҢ…мқҙ лҗ©лӢҲлӢӨ. мқҙлҹ¬н•ң нҠ№м„ұ л•Ңл¬ём—җ лҹ°нғҖмһ„м—җ нғҖмһ…мқ„ мІҙнҒ¬н•ҳлҠ” кІғмқҙ м–ҙл өмҠөлӢҲлӢӨ. мһҗл°”лҠ” м—¬м „нһҲ м ңл„ӨлҰӯ нғҖмһ…мқ„ м¶”лЎ н•ҙм•ј н•ҳм§Җл§Ң, мӮӯм ңк°Җ лҗҳм–ҙ н•„мҡ”н•ң м •ліҙлҘј м–»мқ„ мҲҳ м—ҶмҠөлӢҲлӢӨ. мһҗл°” 8м—җм„ңлҠ” лһҢлӢӨлҘј м§Җмӣҗн•ҳкё° мң„н•ҙ нғҖмһ… м¶”лЎ мқҙ к°ңм„ лҗҳм—ҲлҠ”лҚ° мЎ°кёҲ лҚ” мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

л©”мҶҢл“ң нҳём¶ң мӢң мқёмһҗмқҳ нғҖмһ… м¶”лЎ 

мһҗл°”8 мқҙм „м—җ м ңл„ӨлҰӯ нғҖмһ…мқҳ мқёмһҗлҘј л„ҳкІЁмЈјлҠ” кІҪмҡ° нғҖмһ… м¶”лЎ мқҙ м•ҲлҗҳлҠ” кІҪмҡ°к°Җ мһҲм—ҲмҠөлӢҲлӢӨ.

Collections.emptyList() лҠ” м ңл„ӨлҰӯ нғҖмһ…мқ„ м•Ң мҲҳ м—Ҷкё° л•Ңл¬ём—җ List<Object> нғҖмһ…мңјлЎң кІ°кіјлҘј лҰ¬н„ҙн•ҳкІҢ лҗ©лӢҲлӢӨ. л”°лқјм„ң processNames() мқҳ мқёмһҗлҠ” нғҖмһ…мқҙ л§һм§Җ м•Ҡм•„ м»ҙнҢҢмқј м—җлҹ¬к°Җ лӮ©лӢҲлӢӨ. н•ҳм§Җл§Ң мһҗл°”8м—җм„ң мқҙкІғмқҙ к°ңм„ лҗҳм–ҙ нғҖмһ… мҰқкұ° м—ҶмқҙлҸ„ мқёмһҗмқҳ нғҖмһ…мқ„ мң м¶”н•  мҲҳ мһҲкІҢ лҗҳм—ҲмҠөлӢҲлӢӨ.

м—°мҮ„ л©”мҶҢл“ң нҳём¶ң мӢң мқёмһҗмқҳ нғҖмһ… м¶”лЎ 

emptyList() л©”мҶҢл“ңлҘј нҳём¶ңн•ҳл©ҙм„ң нғҖмһ…мқҙ м ңкұ°лҗҳкё° л•Ңл¬ём—җ м—°мҮ„м ҒмңјлЎң нҳём¶ңлҗҳлҠ” л¶Җл¶„м—җм„ң мқёмһҗлҘј м•Ңм•„мұҢ мҲҳк°Җ м—ҶмҠөлӢҲлӢӨ. мһҗл°” 8м—җм„ң мҲҳм •лҗ  мҳҲм •мқҙм—ҲмңјлӮҳ м·ЁмҶҢлҗҳм–ҙ м—¬м „нһҲ м»ҙнҢҢмқјлҹ¬м—җкІҢ лӘ…мӢңм ҒмңјлЎң нғҖмһ…мқ„ м•Ңл ӨмӨҳм•ј н•©лӢҲлӢӨ.

н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨ

мһҗл°”лҠ” лһҢлӢӨлҘј м§Җмӣҗн•ҳкё° мң„н•ҙм„ң нғҖмһ… м¶”лЎ мқ„ к°•нҷ”н•ҙм•ј н–ҲмҠөлӢҲлӢӨ. к·ёлһҳм„ң 'н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨвҖҷк°Җ лӮҳмҷ”мҠөлӢҲлӢӨ. н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨлҠ” н•ҳлӮҳмқҳ м¶”мғҒ л©”мҶҢл“ң(Single abstract method, лӢЁмқј м¶”мғҒ л©”мҶҢл“ң)лЎң мқҙлЈЁм–ҙм§„ мқён„°нҺҳмқҙмҠӨмқёлҚ°, м—¬кё°м„ң н•ЁмҲҳмқҳ мӢңк·ёлӢҲміҗк°Җ м •мқҳлҗҳм–ҙ мһҲкё° л•Ңл¬ём—җ м»ҙнҢҢмқјлҹ¬к°Җ мқҙ м •ліҙлҘј м°ёкі н•ҙм„ң лһҢлӢӨм—җм„ң мғқлһөлҗң м •ліҙл“Өмқ„ м¶”лЎ н•  мҲҳ мһҲкІҢ лҗ©лӢҲлӢӨ.

@FunctionalInterface

н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨлҠ” лӢЁ н•ҳлӮҳмқҳ л©”мҶҢл“ңлҘј к°Җм§Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ. м»ҙнҢҢмқјлҹ¬к°Җ лҜёлҰ¬ мІҙнҒ¬н•  мҲҳ мһҲлҸ„лЎқ @FunctionalInterface м–ҙл…ён…Ңмқҙм…ҳмңјлЎң н‘ңмӢңн•ҙмӨ„ мҲҳ мһҲмҠөлӢҲлӢӨ. кё°мЎҙ JDK мқҳ Runnable мқҙлӮҳ Callabe к°ҷмқҖ мқён„°нҺҳмқҙмҠӨл“Өмқҙ мқҙ м–ҙл…ён…Ңмқҙм…ҳмңјлЎң к°ңм„ лҗҳм—ҲмҠөлӢҲлӢӨ. лҳҗн•ң лӢӨлҘё мӮ¬мҡ©мһҗм—җкІҢ мқён„°нҺҳмқҙмҠӨмқҳ мқҳлҸ„лҘј м„ӨлӘ…н•ҙмӨ„ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

мғҒмҶҚ

л§Ңм•Ҫ н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨлҘј мғҒмҶҚн•ҳлҠ” кІҪмҡ°м—җлҸ„ мқҙлҹ¬н•ң нҠ№м„ұмқ„ к·ёлҢҖлЎң мқҙм–ҙл°ӣмҠөлӢҲлӢӨ.

лһҢлӢӨмқҳ нғҖмһ… м¶”лЎ 

лһҢлӢӨлҠ” мқёмһҗмқҳ нғҖмһ…мқ„ м¶”лЎ н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мң„м—җм„ң мӮҙнҺҙліё кІғмІҳлҹј н•ЁмҲҳнҳ• мқён„°нҺҳмқҙмҠӨк°Җ нғҖмһ…м—җ лҢҖн•ң м •ліҙлҘј м»ҙнҢҢмқјлҹ¬м—җкІҢ м ңкіөн•ң лҚ•л¶„мһ…лӢҲлӢӨ.

мҳҲмҷё

@FunctionalInterface м—җлҠ” н•ҳлӮҳмқҳ л©”мҶҢл“ңл§Ң мһ‘м„ұн•  мҲҳ мһҲлӢӨкі  н–ҲлҠ”лҚ°, м—¬кё°м—җлҠ” мҳҲмҷёк°Җ мһҲмҠөлӢҲлӢӨ.

  • Object нҒҙлһҳмҠӨмқҳ л©”мҶҢл“ңлҘј мҳӨлІ„лқјмқҙл“ңн•ҳлҠ” кІҪмҡ°
  • л””нҸҙнҠё л©”мҶҢл“ң
  • мҠӨнғңнӢұ л©”мҶҢл“ң

мҳҲлҘј л“Өм–ҙ Comparator мқҳ кІҪмҡ° @FunctionalInterface мқёлҚ° л©”мҶҢл“ңк°Җ л§Һмқҙ мһҲмҠөлӢҲлӢӨ. мӮҙнҺҙліҙл©ҙ л””нҸҙнҠё л©”мҶҢл“ң, мҠӨнғңнӢұ л©”мҶҢл“ң, Object мҳӨлІ„лқјмқҙл“ңн•ң л©”мҶҢл“ңк°Җ мһҲкі  м¶”мғҒ л©”мҶҢл“ңмқҳ кІҪмҡ°лҠ” compare л©”мҶҢл“ң н•ҳлӮҳ лҝҗмһ…лӢҲлӢӨ
